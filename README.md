# findSurf
A full-stack surf atlas app, built with EJS, MongoDB, NodeJS, and bootstrap!

Available at findSurf.maxwellmandina.com

To Run Locally:

Clone Repo
Navigate to findSurf
Open Terminal
run npm install
run node index.js
